# Stitch Prompting Guide: The "Cyber-Bong" Aesthetic

When the AI generates prompts for you to paste into Google Stitch, they will follow these core design rules to ensure consistency across the Bong Bari Comedy website.

## Core Aesthetic Directives to always include in Stitch Prompts:

1.  **Theme Structure**: 
    "Aesthetics: Premium, 'Cyber-Bong' theme, deep dark mode. The background must not be pure black, use very dark zinc (`bg-zinc-950`)."

2.  **Typography**: 
    "Typography: Use modern, bold sans-serif fonts for English headings with tight tracking. Use glowing gradient text (blue to cyan, or red to pink) for primary headings to give a neon effect."

3.  **Components & Cards**:
    "Card Styles: All cards must have rounded corners (`rounded-2xl` or `rounded-xl`), a slight translucent dark background (`bg-zinc-900/50`), and an ultra-thin white border (`ring-1 ring-white/10`) to simulate a glass iPhone UI feeling."

4.  **Layout & Responsiveness**:
    "Layout Rules: Must be completely responsive using fluid grid techniques. Do not use fixed pixel widths for containers. Ensure elements can scale down gracefully to a 320px mobile screen."

5.  **Interactive Elements**:
    "Buttons & Interactivity: Primary buttons should either be vibrant gradients (e.g., Apple Music red) or glassmorphic (`backdrop-blur-md bg-white/10`). Add distinct hover and active states."

## Example of a Good Stitch Prompt:

> *"Generate a mobile-first 'Meet the Team' section. The background should be dark zinc (`bg-zinc-950`). The heading 'Our Comedians' should have a glowing blue-to-purple gradient. Below it, display a fluid grid of profile cards. Each card must have a glassmorphic background (`bg-black/40 backdrop-blur-md`), a subtle white border (`ring-1 ring-white/10`), a rounded circular profile image, and the person's name in bold white text. Ensure it looks premium and app-like on a mobile phone."*
